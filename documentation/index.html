Overview

Welcome to Discord Bot Manager!

The Welcome to Discord Bot Manager is ideal for managing your Discord bot. Whether sending broadcast messages,
banning players with an individual reason or quickly modifying channels. One button click and quickly done.


It is a simple, usable application programmed with the C# programming language using the WinForms Framework.
It was developed in the Visual Studio integrated development environment. For the realization of a communication
between client like the application and the server of Discord, the NuGet package DSharpPlus was used, which has a
number of functions that make it possible to interact with Discord bots.

Warning
The function in the DBM application can lead to the account being banned or blocked if it is used for malicious purposes. I accept no liability whatsoever if all chats, channels or anything else is deleted or changed.
