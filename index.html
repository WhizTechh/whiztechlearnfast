Commands


Every command starts with prefix dbm.
How to write a complete command in the Command Console → prefix.command=value. For example dbm.consoleForeColor=red or dbm.appClose.

Prefix Command Value Info
dbm initBot your bot token Initializes your
discord bot
dbm consoleForeColor red, white, blue, green, yellow, purple, pink, orange, black Changes font color of
command console
dbm consoleBackColor red, white, blue, green, yellow, purple, pink, orange, black Changes background color
of command console
dbm clearfields – Clears all textfields from the application.
The Bot Token text field is not affected by this
dbm appMinimize – Minimizes application
dbm appRestart – Restarts application
dbm appClose – Closes application
